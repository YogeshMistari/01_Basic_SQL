# 01_Basic_SQL

This repository contains fundamental **Oracle SQL queries** that every beginner should know. It includes examples for:

- Creating and dropping tables
- Inserting, updating, and deleting data
- Selecting and filtering rows
- Using string, numeric, and date functions
- Aggregate functions (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`)
- Grouping and ordering data

These queries are designed to help you build a strong foundation in SQL before moving to advanced concepts.
